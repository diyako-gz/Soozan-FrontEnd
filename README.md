## Installation

Follow these steps to set up and run it:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/diyako-gz/Soozan-FrontEnd.git
   ```

3. **Install requirements:**
   ```bash
   npm install
   ```

4. **Take a Smoke Test:**
   ```bash
   npm run tailwind
   ```
